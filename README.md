# 🎯 Face Recognition System using Python

A real-time face recognition system built using **Python**, **OpenCV**, and the **face_recognition** library. This project detects and identifies faces through a webcam feed.

---

## 🚀 Features
- 📸 Real-time face detection
- 🧠 Face recognition using trained encodings
- 👥 Supports multiple users
- 🏷️ Displays names on detected faces
- ⚡ Fast and efficient processing

---

## 🛠️ Tech Stack
- Python
- OpenCV
- face_recognition
- NumPy
